📖 Guia GitHub Pages
🚀 Publicar sua Loja Gratuitamente
1

Crie uma conta no GitHub

Acesse github.com e crie uma conta gratuita se ainda não tiver.

2

Crie um repositório público

Clique em New repository. Dê um nome como alpha-suplementos. Selecione Public. Clique em Create repository.

3

Faça upload dos arquivos

Na página do repositório, clique em Add file → Upload files. Selecione index.html e admin.html. Clique em Commit changes.

4

Ative o GitHub Pages

Vá em Settings → Pages. Em Source, selecione Deploy from a branch. Selecione a branch main e pasta / (root). Clique em Save.

5

Aguarde e acesse

Em alguns minutos, sua loja estará disponível em https://SEU-USUARIO.github.io/alpha-suplementos/. O painel admin fica em .../admin.html.

6

Como atualizar

Para atualizar, repita o passo 3: faça upload do arquivo atualizado. O GitHub substituirá automaticamente.

⚠️ Importante: As configurações do painel admin (produtos, cupons, cores etc.) ficam salvas no localStorage do seu navegador. Se você trocar de computador ou limpar o cache, precisará reconfigurar. Mantenha um backup dos seus produtos via Google Sheets!

🔑 GitHub Token

Para salvar produtos diretamente no repositório, informe seu Personal Access Token.
Fica salvo só no seu navegador — nunca fica no código.

Como gerar:
1. github.com → Settings → Developer settings
2. Personal access tokens → Tokens (classic)
3. Generate new token (classic)
4. Marque o escopo repo → clique Generate
5. Cole o token acima ✅
